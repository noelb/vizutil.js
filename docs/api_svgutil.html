SVGUtil provides a variety of methods for easily generating SVGElements that can be added to an SVG document.

  • SVGUtil . getRect ( document, x, y, width, height, center )

    Name Default Description
    x : float
    y : float
    width : float
    height : float
    center : boolean true If true, the {x,y} parameters define the center point. If false, the {x,y} define the top left corner.
    Return SVGRectElement

    ...

    Example

                    var svgDoc = document.createElementNS("http://www.w3.org/2000/svg", "svg");
                var rect = SVGUtil.getRect(document, 0, 0, 100, 100 );
                svgDoc.appendChild( rect );

  • SVGUtil . getRoundedRect ( document, x, y, width, height, rx, ry, center )

    Name Default Description
    x : float
    y : float
    width : float
    height : float
    rx : float 2 The horizontal radius of the rounded corner
    ry : float null The vertical radius of the rounded corner.
    center : boolean true If true, the {x,y} parameters define the center point. If false, the {x,y} define the top left corner.
    Return SVGRectElement

    ...

    Example

                    // A rounded rect
                // centered at 0,0
                // width/height of 100
                // corners rounded to 10
                var svgDoc = document.createElementNS("http://www.w3.org/2000/svg", "svg");
                var rect = SVGUtil.getRoundedRect(document, 0, 0, 100, 100, 10 );
                svgDoc.appendChild( rect );

  • SVGUtil . getCircle ( document, cx, cy, r )

    Name Description
    cx : float
    cy : float
    r : float
    Return SVGCircleElement

    ...

    Example

                    // Creates a circle centered on 0,0 with a radius of 100

  • SVGUtil . getCross ( document, cx, cy, radius, thickness, numPoints=5 )

    Name Defaults Description
    cx : float
    cy : float
    radius : float
    thickness : float 20
    numPoints : float 4
    Return SVGEllipseElement

    ...

    Example

                    // Creates a cross
                var svgDoc = document.createElementNS("http://www.w3.org/2000/svg", "svg");
                var cross = SVGUtil.getCross(document, 0, 0, 200, 100 );
                svgDoc.appendChild( cross );

                // Creates a snowflake-like shape by setting 6 points instead of the default 4
                var svgDoc = document.createElementNS("http://www.w3.org/2000/svg", "svg");
                var snowflake = SVGUtil.getCross(document, 0, 0, 200, 100, 6 );
                svgDoc.appendChild( snowflake );
