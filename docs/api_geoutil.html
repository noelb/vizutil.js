References to "Point" objects can either be in the format of a generic object (e.g., {x:12,y:13}) or a numeric array (e.g., [12,13]). The array-style is the default. For some methods, you must specify which you want using the "xy" boolean parameter.

  • GeoUtil . toRad ( degrees )

    Name Default Description
    degrees : float An angle in degree measure
    Return float An angle in radian measure

    Example

                console.log( GeomUtil.toRad(180) );

            >

  • GeoUtil . toDeg ( radians )

    Name Default Description
    radians : float An angle in radian measure
    Return float an angle in degree measure

    Example

                console.log( GeomUtil.toDeg(3.14) );

            > 180

  • GeoUtil . mercProject ( lat, lon, sizeX, sizeY, xy )

    Name Default Description
    lat, lon : float A geoposition
    sizeX, sizeY : float The width and height of the rectangle to project onto
    xy : boolean false Determine the format of the point - [] vs. {}
    Return Point

    Example

                console.log( GeomUtil.mercProject(40.712775,-74.005973) );
            >

            console.log( GeomUtil.mercProject(40.712775,-74.005973, 500, 500) );
            >

  • GeoUtil . invMercProject ( ptx, pty, sizeX, sizeY )

    Name Description
    ptx, pty : float A position within a rectangle
    sizeX, sizeY : float The width and height of the rectangle
    Return [float,float] The latitude and longitude [lat,lon] of the point on a globe

    Example

                console.log( GeomUtil.invMercProject(.5,.5) );
            >

            console.log( GeomUtil.mercProject(130, 150, 500, 500) );
            >

  • GeoUtil . r ()

    Name Description
    Return float Always returns 6371, which is the radius of the earth in km

  • GeoUtil . distance ( lat0, lon0, lat1, lon1 )

    Name Description
    lat0, lon0 : float Start position
    lat1, lon1 : float End position
    Return float Distance between the two positions.

  • GeoUtil . bearing ( lat0, lon0, lat1, lon1 )

    Name Description
    lat0, lon0 : float Start position
    lat1, lon1 : float End position
    Return float Direction of travel in radians.
